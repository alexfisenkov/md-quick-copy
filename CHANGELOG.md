# Changelog

## 1.2.0 - 2026-05-30

- Added visible table copy controls in Quick Look previews: Markdown, CSV, and TSV.
- Added a generated document outline for Markdown files with multiple headings.
- Added a useful host app window with install status, GitHub access, and maintenance commands.
- Added Core tests for table export and outline generation.
- Preserved identifier underscores when exporting tables as plain CSV/TSV text.
- Added GitHub Actions CI for project generation, tests, and Release app build.
- Updated public documentation and the local engineering report.

## 1.1.0 - 2026-05-30

- Expanded Markdown rendering beyond fenced code blocks.
- Added GitHub-Flavored Markdown table rendering with column alignment.
- Added tilde fences, indented code blocks, and exported language-label code detection.
- Added support for headings, lists, task lists, blockquotes, and horizontal rules.

## 1.0.0 - 2026-05-30

- Added a native macOS Quick Look preview extension for `.md` and `.markdown` files.
- Added visible copy buttons for fenced code blocks.
- Added local build and install scripts for `/Applications`.
